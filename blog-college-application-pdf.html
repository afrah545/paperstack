How to Compress a PDF for College Applications

6 min read · Updated June 2026

You're a few clicks from submitting your application, and the portal throws an error: "File exceeds maximum upload size." It's almost always the transcript — a scanned copy from your school, often saved at a resolution way higher than anyone actually needs to read it.

This happens constantly during application season, and the fix takes under a minute once you know which tool to reach for.

When you'd actually need this

  • Scanned transcripts — school-issued transcripts scanned at high resolution are the most common culprit
  • Recommendation letters — combined or scanned letters that add up in size
  • Portfolio submissions — art, design, or writing portfolios with multiple images embedded
  • Supplemental documents — financial aid forms, certificates, or supporting paperwork

Typical upload limits

PlatformTypical limit
Common App document uploadsAround 5 MB per document (check the specific field)
University-specific portalsVaries widely, often 2–10 MB
Scholarship application platformsOften 2–5 MB
Email submission (where allowed)20–25 MB, limited by the recipient's email provider

Always check the specific limit listed on the portal you're using — these vary by institution and platform, and the numbers above are typical ranges, not guarantees.

Which compression level to use for application documents

🟢

Light

Best when every detail matters — a portfolio piece where image quality is part of what's being judged.

🟡

Recommended

The right default for transcripts and letters — text stays crisp, file size drops significantly.

🔴

Maximum

Use only if you're right at the size limit and the document is plain text — admissions readers need to read it, not admire it.

If it's still too large after compressing

  • Ask for a digital original instead of scanning a printed copy — many schools can email a transcript directly as a PDF, which starts out far smaller than a phone-scanned version of a printed page.
  • Combine documents only where the portal asks for one file — if recommendation letters and a transcript need to go in as a single upload, merge them first, then compress the combined file once rather than compressing each separately.
  • Re-scan at a lower resolution — if you're scanning yourself, 200 DPI is plenty for a readable text document and produces a much smaller starting file than 300+ DPI.
💡 Quick tip Compress and check the file size before the night your application is due — portal errors are far more stressful with a deadline closing in an hour than with a few days to spare.

Common questions

Q.Will compressing make my transcript hard for admissions officers to read?
At Light or Recommended compression, no — text stays sharp and fully legible. Compression mainly affects embedded images, not text clarity, so a text-based transcript holds up well even at higher compression.
Q.Can I combine my transcript and recommendation letters into one PDF before uploading?
Yes — if a portal field asks for a single file, use a Merge PDF tool to combine them first, then compress the merged result once.
Q.Is it safe to compress documents with my personal academic records?
Yes — PaperStack compresses your file entirely inside your browser. It's never uploaded to a server, so nobody but you ever sees the content.
Q.My file is still too big even at Maximum compression — now what?
This usually means the original scan was at a very high resolution or contains large embedded images. Try re-scanning at a lower DPI, or contact the institution that issued the document and ask if they can provide a smaller digital copy directly.
